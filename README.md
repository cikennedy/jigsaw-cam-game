# Puzzle Camera Game
  ![GitHub License](https://img.shields.io/badge/license-MIT-yellow.svg)

  ## Description

  A game that creates a jigsaw puzzle using the camera on a device. Uses only HTML, vanilla JavaScript, MySQL, and PHP.
